在台式机上安装chrome os的注意事项

2020-12-15

本文所使用的安装教程为少数派的一篇文章

旧电脑再利用：完整版 Chrome OS 安装指南
另外在youtube上也有几个教程，最原始的似乎是Kedar Nimbalkar的教程，你也可以搜到中文的教程视频，对于在一块硬盘上装双系统有详细的介绍。

https://www.youtube.com/watch?v=tzFCipCeXFk

https://www.youtube.com/watch?v=HGH96fXk4p0&t=565s

安装注意事项：

1、个人使用的是台式机，赛扬g18xx(具体型号忘了，四代cpu)，按照教程装完之后开机一直重启，后来换了个赛扬g3930(好像是这个型号，七代cpu)才成功启动。(似乎没有关闭安全启动，但主板在折腾过程中报废，已无法验证)

2、youtube上的中文教程用的都是配置好的install.sh文件，并且都告诉你要科学上网(用刷过固件能科学上网的路由器)，实际上安装的时候并不需要科学上网，按照少数派的文章，使用ubuntu作为引导盘最好，Linux Mint我没有用过，大概是apt update的时候使用的源有的在国外，导致下载速度极慢，所以才清一色的要求要科学上网，我个人使用ubuntu的时候速度正常。

3、不需要换源，反正我没换，速度正常。

4、少数派给的install.sh可以进行简化

sudo apt update

sudo apt install figlet pv cgpt

sudo bash chromeos-install.sh -src rammus_recovery.bin -dst /dev/sda

手动输入最好

其中第三行最后的sda，意思是硬盘a(第一块硬盘)，在使用ubuntu引导盘时可以使用

sudo fdisk -l

来查看你要安装的硬盘是哪一个。(如果使用虚拟机linux的话，fdisk可能会无法使用)

5、不需要sudo apt upgrade

6、下载谷歌chrome os恢复镜像的网站，有时候无法打开，等一段时间就行了，你也可以选择使用网页快照网站(会对网站的历史页面进行保存，不过要是网站服务器没开也没折)

7、不推荐在笔记本上折腾，问就是驱动问题
