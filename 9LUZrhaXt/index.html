2023-08-20

如何在国内下载谷歌云盘文件

首先获取一个谷歌云盘的链接 例如https://drive.google.com/file/d/1eUh7J6WOEujLrQSBO1gV6tpbLMvzIkRF/view?usp=sharing
将其修改为 https://drive.google.com/uc?id=1eUh7J6WOEujLrQSBO1gV6tpbLMvzIkRF

然后租一个国外vps，例如vultr
选择debian10（不要选12，不知道哪里有问题，无法继续）
安装git pip

git clone https://github.com/wkentaro/gdown.git
cd gdown
pip install gdown

下载文件gdown https://drive.google.com/uc?id=1eUh7J6WOEujLrQSBO1gV6tpbLMvzIkRF

下载完成后使用xftp（随便什么都可以，我用的这个）链接vps下载就行
