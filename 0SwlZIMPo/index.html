DELL 7050 MFF 黑苹果

2021-05-07

配置:CPU i5-6400，intel ax200

按照官方网站配置
https://dortania.github.io/OpenCore-Install-Guide/
OC 0.6.9
全程按照官方教程配置，结果没声音，最终选择万能淘宝解决。
需要修改的点就两个
1、ACPI-->>--patch
需要额外添加第三行的数据(前两个没启用)

2、DeviceProperties-->>--add
需要添加第一行以及右边的数据
(按照教程这个算是最终固定音频的步骤，我直接一步到位了，alcid的调试之前没成功，这个是从github上其他人的7050的EFI上直接搬过来的，上面ACPI的数据其实别人的EFI上也有)

无线 OK
HDMI OK
BIOS设置使用的这位老哥的方案
https://github.com/linkev/Dell-Optiplex-7050-Micro-Hackintosh/blob/master/BIOS.md

其他功能因为没有需求，暂不清楚

8月15日更新
OC更新到0.7.2，由于我没有升级到big sur，而是一直使用Catalina，因此需要修改配置
opencore configurator内修改：
Misc--security--SecureBootModel--j137
UEFI--APFS--MinVersion--无限制 or Catalina

( 手动修改 MinDate -1 MinVersion -1 或
MinDate 20200306 MinVersion 1412101001000000 )

8月16日更新
没忍住，试着更新Big Sur，直接更新到11.5.2，将上面两个选项修改为默认后开机正常，Wi-Fi无法使用，最后发现Wi-Fi的kexts文件每个大版本各一个，更换后正常，风扇声音比之前感觉有点大

2022年3月13日更新
修改layout-id为12(可使用麦克风）
